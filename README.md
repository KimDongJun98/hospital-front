## 🔧 Stack

**Frontend**
- **Language** : TypeScript
- **Environment** : Next.js
- **Library & Framework** : React, Styled-Components, Axios
- **Deploy** : Vercel

<br/>

**Backend**
- **Language** : TypeScript
- **Environment** : Node.js
- **Library & Framework** : Express
- **Database** : MongoDB
- **Deploy** : Fly.io, MongoDB Atlas(database)
